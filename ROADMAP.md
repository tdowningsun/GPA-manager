# GPA Manager Roadmap

# 繁體中文

GPA Manager 是一個面向大學生的 GPA 管理與學業規劃工具。

本專案希望從單純的 GPA 計算工具開始，逐步發展成協助學生管理課程資料、分析學業狀況以及規劃學習目標的工具。

---

# 目前版本

## v3.1.0 - 穩定桌面版本

已完成：

- 完成 Python 桌面圖形化應用程式
- 支援 Windows 高 DPI 顯示
- 支援多門課程管理
- 支援新增課程
- 支援刪除指定課程
- 支援清空全部課程
- 課程資料變更後自動更新 GPA
- 建立完整 GitHub 專案文件

# English

GPA Manager is a GPA management and academic planning tool designed for university students.

The project aims to evolve from a simple GPA calculator into a practical application that helps students manage academic data, analyze performance, and plan learning goals.

---
