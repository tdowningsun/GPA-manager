
# GPA Manager

## 繁體中文

GPA Manager 是一個使用 Python 開發的命令列（CLI）應用程式，用於根據課程學分與成績計算大學 GPA。

本專案強調簡潔性、正確性與清晰的程式結構。

### 主要功能
- 透過終端機輸入多門課程
- 支援數字與文字形式的成績輸入
- 使用固定規則將成績轉換為績點
- 自動計算加權 GPA
- 以表格形式輸出結果

### 專案結構
- main.py：負責使用者互動與流程控制
- gpa_calculator.py：負責 GPA 計算邏輯
- data.py：存放成績規則與對照表

### 專案目的
本專案用於學習 Python 基礎語法與基本軟體工程概念，例如模組化設計與職責分離。

## English

GPA Manager is a Python command-line (CLI) application designed to help university students calculate their Grade Point Average (GPA) based on course credits and scores.

It focuses on simplicity, correctness, and clean software structure.

### Key Features
- Add multiple courses via terminal input
- Support numeric and text-based score input
- Convert scores into grade points using a fixed grading system
- Calculate weighted GPA automatically
- Display results in a clean table format

### Project Structure
- main.py: Handles user interaction and program flow
- gpa_calculator.py: Contains GPA calculation logic
- data.py: Stores grading rules and mappings

### Purpose
This project is built for learning Python fundamentals and practicing basic software engineering principles such as modular design and code separation.
