# GPA Manager

A desktop GPA management and academic planning tool built with Python.

GPA Manager helps university students calculate, manage, and analyze their GPA through an intuitive graphical interface.

The project focuses on clean architecture, usability, and practical student needs.

# 繁體中文

## 簡介

GPA Manager 是一個使用 Python 開發的桌面 GPA 管理工具。

本專案旨在幫助大學生快速計算 GPA、管理課程成績，並逐步發展為學業規劃與研究生申請輔助工具。

專案強調：

- 清晰的程式架構
- 良好的使用體驗
- 可持續擴展性

---

## 主要功能

### GPA 計算

- 支援多門課程輸入
- 根據學分與成績計算加權 GPA
- 支援數字與文字形式成績輸入
- 自動生成計算結果

### 桌面 GUI

- 基於 Python Tkinter 開發
- 支援高 DPI 螢幕顯示
- 提供直觀操作介面
- 使用表格管理課程資料

### 課程管理

- 新增課程
- 刪除指定課程
- 清空全部課程
- 修改後自動更新 GPA

# English

## Introduction

GPA Manager is a Python-based desktop GPA management tool designed for university students.

It helps students calculate and manage their GPA through an intuitive graphical interface.

The project focuses on:

- Clean software architecture
- User experience
- Long-term extensibility

---

## Features

### GPA Calculation

- Support multiple courses
- Calculate weighted GPA based on credits and scores
- Support numerical and text-based grade input
- Generate calculation results automatically

### Desktop GUI

- Built with Python Tkinter
- High DPI display support
- User-friendly graphical interface
- Course management table

### Course Management

- Add courses
- Delete selected courses
- Clear all courses
- Automatically update GPA after changes
