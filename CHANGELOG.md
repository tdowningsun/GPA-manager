# Changelog

GPA Manager 的版本更新紀錄。

This file documents all notable changes of GPA Manager.

---

# 繁體中文

## v3.1.0

### 新增

- 新增桌面 GUI 課程管理功能
- 新增刪除指定課程功能
- 新增清空全部課程功能
- 課程修改後自動更新 GPA

### 改進

- 改善課程列表與內部資料同步
- 改善 GUI 使用體驗
- 支援 Windows 高 DPI 螢幕顯示

---

## v3.0.0

### 新增

- 使用 Tkinter 建立圖形化使用者介面
- 新增課程列表表格顯示
- 新增 GUI GPA 計算流程

---

## v2.1.0

### 改進

- 改善命令列操作體驗
- 優化輸入提示
- 新增課程輸入完成後的摘要顯示

---

## v2.0.0

### 修改

- 重新設計課程輸入流程
- 分離課程名稱、學分與成績輸入階段
- 提升程式結構與可維護性

---

## v1.0.0

### 初始版本

- 完成基本 GPA 計算功能
- 支援課程輸入
- 支援加權 GPA 計算
- 建立成績轉換規則

---

# English

## v3.1.0

### Added

- Added desktop GUI course management features
- Added delete selected course function
- Added clear all courses function
- Added automatic GPA update after course changes

### Improved

- Improved synchronization between course table and internal data
- Improved GUI user experience
- Added Windows high DPI display support

---

## v3.0.0

### Added

- Introduced graphical user interface using Tkinter
- Added course table display
- Added GUI-based GPA calculation workflow

---

## v2.1.0

### Improved

- Improved CLI interaction experience
- Improved input prompts
- Added course summary display

---

## v2.0.0

### Changed

- Redesigned course input workflow
- Separated course name, credit, and score input steps
- Improved code structure and maintainability

---

## v1.0.0

### Initial Release

- Implemented basic GPA calculation
- Added course input
- Added weighted GPA calculation
- Added grading conversion rules
